# R B Constructions – Website

A full-stack professional website for R B Constructions, Lakhanpur, Kathua, J&K.

## Tech Stack
- **Backend**: Node.js + Express
- **Frontend**: HTML5, CSS3, Vanilla JavaScript
- **Fonts**: Bebas Neue (display), Barlow Condensed, Barlow (body)

## Setup & Run

### Prerequisites
- Node.js v16 or higher
- npm

### Installation

```bash
# 1. Navigate to project directory
cd rb-constructions

# 2. Install dependencies
npm install

# 3. Start the server
npm start
```

The site will be live at: **http://localhost:3000**

### Development (with auto-restart)
```bash
npm install -g nodemon
npm run dev
```

## Features
- ✅ Responsive design (mobile, tablet, desktop)
- ✅ Heavy vehicle rental — Transit Mixer (₹400/m³), Bulker (₹3,500/ton), Excavator (₹3,000/hr)
- ✅ 24-item construction inventory with category filters & search
- ✅ Inquiry/booking modal for vehicles and materials
- ✅ Contact form with backend API
- ✅ Animated counters and scroll-reveal effects
- ✅ REST API: `/api/vehicles`, `/api/inventory`, `/api/inquiry`, `/api/stats`
